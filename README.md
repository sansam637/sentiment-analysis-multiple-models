# Sentiment Analysis with Multiple Models

## Description
This project compares Naive Bayes and SVM for sentiment analysis using TF-IDF.

## How to Run
pip install -r requirements.txt
python main.py

## Results
SVM performs better than Naive Bayes.

The model generates the following outputs:

- Accuracy comparison graph → results/accuracy_graph.png
- Confusion matrix → results/confusion_matrix.png
